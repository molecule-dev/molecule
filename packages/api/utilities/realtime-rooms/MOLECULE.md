# @molecule/api-realtime-rooms

Realtime rooms abstraction: named pub/sub rooms with membership, capacity, join codes, role-based auth, built atop @molecule/api-realtime.

## Type
`utility`

## Injection Notes

### Requirements
- None

### Post-Injection Steps
- Run `npm install` to install dependencies
- Run `npm run build` to compile

- None yet
