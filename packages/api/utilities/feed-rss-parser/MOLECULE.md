# @molecule/api-feed-rss-parser

RSS/Atom/JSON Feed parser — normalizes RSS 2.0, Atom 1.0, RDF 1.0, and JSON Feed into a uniform { feed, items[] } shape.

## Type
`utility`

## Injection Notes

### Requirements

### Post-Injection Steps
- Run `npm install` to install dependencies
- Run `npm run build` to compile
