# @molecule/api-locales-scheduler

Translations for molecule scheduler package in 79 languages

## Purpose

Provides translations for the `@molecule/api-scheduler` package which has 1 translation key.

## Languages

79 languages supported: af, am, ar, az, be, bg, bn, bs, ca, cs, cy, da, de, el, en, es, et, eu, fa, fi, fil, fr, ga, gl, gu, ha, he, hi, hr, hu, hy, id, ig, is, it, ja, ka, kk, km, kn, ko, ky, lo, lt, lv, mk, ml, mn, mr, ms, mt, my, nb, ne, nl, pa, pl, pt, ro, ru, si, sk, sl, sq, sr, sv, sw, ta, te, th, tr, uk, ur, uz, vi, yo, zh, zh-TW, zu.

## Usage

```typescript
import { af, am, ar } from '@molecule/api-locales-scheduler'
import type { SchedulerTranslationKey, SchedulerTranslations } from '@molecule/api-locales-scheduler'
```

## Translation Keys

| Key | English |
|-----|---------|
| `scheduler.error.noProvider` | Scheduler provider not configured. Call setProvider() first. |
