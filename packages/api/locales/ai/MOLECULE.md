# @molecule/api-locales-ai

Translations for molecule AI package in 79 languages

## Purpose

Provides translations for the `@molecule/api-ai` package which has 3 translation keys.

## Languages

79 languages supported: af, am, ar, az, be, bg, bn, bs, ca, cs, cy, da, de, el, en, es, et, eu, fa, fi, fil, fr, ga, gl, gu, ha, he, hi, hr, hu, hy, id, ig, is, it, ja, ka, kk, km, kn, ko, ky, lo, lt, lv, mk, ml, mn, mr, ms, mt, my, nb, ne, nl, pa, pl, pt, ro, ru, si, sk, sl, sq, sr, sv, sw, ta, te, th, tr, uk, ur, uz, vi, yo, zh, zh-TW, zu.

## Usage

```typescript
import { af, am, ar } from '@molecule/api-locales-ai'
import type { AiTranslationKey, AiTranslations } from '@molecule/api-locales-ai'
```

## Translation Keys

| Key | English |
|-----|---------|
| `ai.error.noProvider` | AI provider not configured. Bond an AI provider first. |
| `ai.error.apiError` | AI API request failed. |
| `ai.error.noResponseBody` | AI response body is empty. |
