# @molecule/app-locales-icons

Translations for @molecule/app-icons in 79 languages

## Purpose

Provides translations for the `@molecule/app-icons` package which has 2 translation keys.

## Languages

79 languages supported: af, am, ar, az, be, bg, bn, bs, ca, cs, cy, da, de, el, en, es, et, eu, fa, fi, fil, fr, ga, gl, gu, ha, he, hi, hr, hu, hy, id, ig, is, it, ja, ka, kk, km, kn, ko, ky, lo, lt, lv, mk, ml, mn, mr, ms, mt, my, nb, ne, nl, pa, pl, pt, ro, ru, si, sk, sl, sq, sr, sv, sw, ta, te, th, tr, uk, ur, uz, vi, yo, zh, zh-TW, zu.

## Usage

```typescript
import { af, am, ar } from '@molecule/app-locales-icons'
import type { IconsTranslationKey, IconsTranslations } from '@molecule/app-locales-icons'
```

## Translation Keys

| Key | English |
|-----|---------|
| `icons.error.noIconSet` | No IconSet has been set. Call setIconSet() at app startup with an icon library (e.g., @molecule/app-icons-molecule). |
| `icons.error.notFound` | Icon  |
