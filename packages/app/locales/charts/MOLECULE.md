# @molecule/app-locales-charts

Translations for @molecule/app-charts in 79 languages

## Purpose

Provides translations for the `@molecule/app-charts` package which has 2 translation keys.

## Languages

79 languages supported: af, am, ar, az, be, bg, bn, bs, ca, cs, cy, da, de, el, en, es, et, eu, fa, fi, fil, fr, ga, gl, gu, ha, he, hi, hr, hu, hy, id, ig, is, it, ja, ka, kk, km, kn, ko, ky, lo, lt, lv, mk, ml, mn, mr, ms, mt, my, nb, ne, nl, pa, pl, pt, ro, ru, si, sk, sl, sq, sr, sv, sw, ta, te, th, tr, uk, ur, uz, vi, yo, zh, zh-TW, zu.

## Usage

```typescript
import { af, am, ar } from '@molecule/app-locales-charts'
import type { ChartsTranslationKey, ChartsTranslations } from '@molecule/app-locales-charts'
```

## Translation Keys

| Key | English |
|-----|---------|
| `charts.placeholder.label` | {{type}} chart |
| `charts.placeholder.description` | Use a proper chart provider |
