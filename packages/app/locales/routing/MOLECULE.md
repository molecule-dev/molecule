# @molecule/app-locales-routing

Translations for @molecule/app-routing in 79 languages

## Purpose

Provides translations for the `@molecule/app-routing` package which has 3 translation keys.

## Languages

79 languages supported: af, am, ar, az, be, bg, bn, bs, ca, cs, cy, da, de, el, en, es, et, eu, fa, fi, fil, fr, ga, gl, gu, ha, he, hi, hr, hu, hy, id, ig, is, it, ja, ka, kk, km, kn, ko, ky, lo, lt, lv, mk, ml, mn, mr, ms, mt, my, nb, ne, nl, pa, pl, pt, ro, ru, si, sk, sl, sq, sr, sv, sw, ta, te, th, tr, uk, ur, uz, vi, yo, zh, zh-TW, zu.

## Usage

```typescript
import { af, am, ar } from '@molecule/app-locales-routing'
import type { RoutingTranslationKey, RoutingTranslations } from '@molecule/app-locales-routing'
```

## Translation Keys

| Key | English |
|-----|---------|
| `routing.error.missingParam` | Missing param  |
| `routing.error.routeNotFound` | Route  |
| `routing.error.useMoleculeRouterOutsideProvider` | useMoleculeRouter must be used within a MoleculeRouterProvider |
