# @molecule/app-locales-ai-chat

Translations for molecule AI chat components in 79 languages

## Purpose

Provides translations for the `@molecule/app-ai-chat` package which has 4 translation keys.

## Languages

79 languages supported: af, am, ar, az, be, bg, bn, bs, ca, cs, cy, da, de, el, en, es, et, eu, fa, fi, fil, fr, ga, gl, gu, ha, he, hi, hr, hu, hy, id, ig, is, it, ja, ka, kk, km, kn, ko, ky, lo, lt, lv, mk, ml, mn, mr, ms, mt, my, nb, ne, nl, pa, pl, pt, ro, ru, si, sk, sl, sq, sr, sv, sw, ta, te, th, tr, uk, ur, uz, vi, yo, zh, zh-TW, zu.

## Usage

```typescript
import { af, am, ar } from '@molecule/app-locales-ai-chat'
import type { ChatTranslationKey, ChatTranslations } from '@molecule/app-locales-ai-chat'
```

## Translation Keys

| Key | English |
|-----|---------|
| `chat.error.httpError` | HTTP {{status}}: {{text}} |
| `chat.error.noResponseBody` | No response body |
| `chat.error.streamError` | Stream error |
| `chat.error.unknownError` | Unknown error |
