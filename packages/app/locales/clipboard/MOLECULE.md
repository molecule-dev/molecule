# @molecule/app-locales-clipboard

Translations for @molecule/app-clipboard in 79 languages

## Purpose

Provides translations for the `@molecule/app-clipboard` package which has 2 translation keys.

## Languages

79 languages supported: af, am, ar, az, be, bg, bn, bs, ca, cs, cy, da, de, el, en, es, et, eu, fa, fi, fil, fr, ga, gl, gu, ha, he, hi, hr, hu, hy, id, ig, is, it, ja, ka, kk, km, kn, ko, ky, lo, lt, lv, mk, ml, mn, mr, ms, mt, my, nb, ne, nl, pa, pl, pt, ro, ru, si, sk, sl, sq, sr, sv, sw, ta, te, th, tr, uk, ur, uz, vi, yo, zh, zh-TW, zu.

## Usage

```typescript
import { af, am, ar } from '@molecule/app-locales-clipboard'
import type { ClipboardTranslationKey, ClipboardTranslations } from '@molecule/app-locales-clipboard'
```

## Translation Keys

| Key | English |
|-----|---------|
| `clipboard.error.noProvider` | @molecule/app-clipboard: No provider set. Call setProvider() with a ClipboardProvider implementation (e.g., from @molecule/app-clipboard-capacitor). |
