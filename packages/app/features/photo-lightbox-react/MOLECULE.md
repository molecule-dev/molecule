# @molecule/app-photo-lightbox-react

Fullscreen photo viewer.

Exports `<PhotoLightbox>` and `LightboxPhoto` type.

## Type
`feature`

## Installation
```bash
npm install @molecule/app-photo-lightbox-react
```
