# @molecule/app-search-autocomplete-react

Search input with typeahead suggestions.

Exports `<SearchAutocomplete>` and `SuggestionItem` type.

## Type
`feature`

## Installation
```bash
npm install @molecule/app-search-autocomplete-react
```
