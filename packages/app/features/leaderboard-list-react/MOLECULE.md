# @molecule/app-leaderboard-list-react

Leaderboard list container.

Exports `<LeaderboardList>` — wraps a stack of `<LeaderboardRow>`s with title and actions.

## Type
`feature`

## Installation
```bash
npm install @molecule/app-leaderboard-list-react
```
