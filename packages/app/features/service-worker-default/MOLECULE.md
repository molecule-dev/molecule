# @molecule/app-service-worker-default

Default workbox-based service worker setup: precache manifest, app-shell navigation routing, image cache with StaleWhileRevalidate, SKIP_WAITING handler. Extracts 48 lines × 123 fleet apps of byte-identical SW boilerplate.
