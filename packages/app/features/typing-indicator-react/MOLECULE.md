# @molecule/app-typing-indicator-react

Three-dot typing indicator for chat UIs.

Exports `<TypingIndicator>` — CSS-only three-dot pulse animation.

## Type
`feature`

## Installation
```bash
npm install @molecule/app-typing-indicator-react
```
