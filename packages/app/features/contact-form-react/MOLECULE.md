# @molecule/app-contact-form-react

Generic contact form (name + email + message + extras).

Exports `<ContactForm>` and `ContactFormValues` type.

## Type
`feature`

## Installation
```bash
npm install @molecule/app-contact-form-react
```

- `react` ^18.0.0 || ^19.0.0
