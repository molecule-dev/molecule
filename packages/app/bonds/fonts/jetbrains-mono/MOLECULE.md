# @molecule/app-fonts-jetbrains-mono

JetBrains Mono font for molecule.dev.

## Type
`provider`

## Installation
```bash
npm install @molecule/app-fonts-jetbrains-mono @molecule/app-fonts
```

## API


Implements `@molecule/app-fonts` interface.
