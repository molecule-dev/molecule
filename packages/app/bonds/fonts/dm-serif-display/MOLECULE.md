# @molecule/app-fonts-dm-serif-display

DM Serif Display font for molecule.dev.

## Type
`provider`

## Installation
```bash
npm install @molecule/app-fonts-dm-serif-display @molecule/app-fonts
```

## API

### Constants

#### `font`

DM Serif Display font definition — local, self-hosted (no external CDN).

```typescript
const font: FontDefinition
```

## Core Interface
Implements `@molecule/app-fonts` interface.

## Injection Notes

### Requirements

Peer dependencies:
- `@molecule/app-fonts` ^1.0.0

### Runtime Dependencies

- `@molecule/app-fonts`
