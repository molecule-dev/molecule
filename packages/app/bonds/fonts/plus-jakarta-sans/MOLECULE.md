# @molecule/app-fonts-plus-jakarta-sans

Plus Jakarta Sans font for molecule.dev.

## Type
`provider`

## Installation
```bash
npm install @molecule/app-fonts-plus-jakarta-sans @molecule/app-fonts
```

## API

### Constants

#### `font`

Plus Jakarta Sans font definition — local, self-hosted (no external CDN).

```typescript
const font: FontDefinition
```

## Core Interface
Implements `@molecule/app-fonts` interface.

## Injection Notes

### Requirements

Peer dependencies:
- `@molecule/app-fonts` ^1.0.0

### Runtime Dependencies

- `@molecule/app-fonts`
