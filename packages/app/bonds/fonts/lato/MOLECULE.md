# @molecule/app-fonts-lato

Lato font for molecule.dev.

## Type
`provider`

## Installation
```bash
npm install @molecule/app-fonts-lato @molecule/app-fonts
```

## API

### Constants

#### `font`

Lato font definition — local, self-hosted (no external CDN).

```typescript
const font: FontDefinition
```

## Core Interface
Implements `@molecule/app-fonts` interface.

## Injection Notes

### Requirements

Peer dependencies:
- `@molecule/app-fonts` ^1.0.0

### Runtime Dependencies

- `@molecule/app-fonts`
