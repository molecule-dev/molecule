# @molecule/app-fonts-source-serif-4

Source Serif 4 font for molecule.dev.

## Type
`provider`

## Installation
```bash
npm install @molecule/app-fonts-source-serif-4 @molecule/app-fonts
```

## API

### Constants

#### `font`

Source Serif 4 font definition — local, self-hosted (no external CDN).

```typescript
const font: FontDefinition
```

Implements `@molecule/app-fonts` interface.
